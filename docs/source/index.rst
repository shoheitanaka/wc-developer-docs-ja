Welcome to WooCommerce 開発者向けドキュメント
===================================

**WooCommerce** (ウーコマース)は WordPress のプラグインで WordPress にネットショップを簡単に構築する事を可能したプラグインです。
`WooCommerce <https://ja.wordpress.org/plugins/woocommerce/>`_

このプラグインではネットショップの基本的な機能を実装しているので、拡張機能を増やす場合は拡張プラグインを開発もしくは導入する必要があります。

公式サイトである`WooCommerce <https://woocommerce.com/>`_では、さまざまな拡張プラグインが販売紹介されています。このサイトはマーケットプレイスの形式になっており、開発して販売することが可能になります。
そこで、このサイトで販売するためのハンドブックがあるので、それの日本語翻訳を作ることにしました。参考にご覧ください。

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.
